About This Site

Welcome to CineLibrary, a movie discovery web app built by me, Praneetha Sai . This project helps users explore movies, view details, posters, and OTT platforms.

My Goal

The goal of this project is to create a clean and modern movie browsing experience, similar to OTT platforms. It focuses on usability, design, and real-time API data.

Who Built This?

This website was designed and developed by me, Praneetha Sai , a Computer Science student passionate about building modern web applications.
